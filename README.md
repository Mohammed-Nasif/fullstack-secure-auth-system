# Full Stack Secure Authentication System

A production-ready authentication system built with NestJS, MongoDB, React, and TypeScript.

## Quick Start

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run start:dev
```

### Frontend  
```bash
cd frontend
npm install
npm run dev
```
